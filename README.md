# Expense Tracker

This is a Flask-based Expense Tracker project built using Python and Flask.

## Features
- Add expenses
- View expenses
- User login/signup

## How to Run This Project

### Prerequisites
- Python installed
- Git installed

### Steps to Run

1. Clone the repository
   git clone https://github.com/Raoyalrajesh123/expense-tracker.git

2.
   cd expense-tracker

3.
   pip install -r requirements.txt

4. 
   python app.py

5. Open browser and visit
   http://127.0.0.1:5000
